# hg19 annotations

This is a collection of annotations I have generated and that can be used for genomic and epigenomic analysis.  

* [**hg19_centromeres.bed**](hg19_centromeres.bed) - hg19 centromeres.
* [**hg19_chromosomes_centers.bed**](hg19_chromosomes_centers.bed) - hg19 chromosome centers.
* [**hg19_chromosomes_proximal_half.bed**](hg19_chromosomes_proximal_half.bed) - hg19 proximal halves of both p and q arms of chromosomes.
* [**hg19_extended_subtelomeres.bed**](hg19_extended_subtelomeres.bed) - hg19 subtelomeric regions directly adjacent to telomeres (this annotation does not contain any gap between telomeres and subtelomeric regions).
* [**hg19_pericentromeric_regions.bed**](hg19_pericentromeric_regions.bed) - hg19 pericentromeric regions that have been defined based on satellites and transposable elements content of chromosomes.
* [**hg19_subtelomeres.bed**](hg19_subtelomeres.bed) - hg19 
* [**hg19_telomeres.bed**](hg19_telomeres.bed) - hg19 subtelomeric regions with gaps remaining between the regions defined and telomeres (not recommended for use).
